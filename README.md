Essa atividade consiste em um site temático inspirado na obra New Moon, parte da saga The Twilight Saga.
A página apresenta informações sobre a história, personagens e elementos visuais relacionados ao universo da narrativa.

O site foi desenvolvido utilizando HTML, CSS e JavaScript, com foco em criar uma experiência visual imersiva baseada em tons escuros e azulados que remetem à atmosfera noturna da história.

Objetivo do Projeto

O objetivo deste projeto é praticar conceitos de desenvolvimento front-end, incluindo:

Estruturação de páginas com HTML

Estilização avançada com CSS

Uso de variáveis de cores

Layout responsivo

Manipulação de elementos com JavaScript

Navegação dinâmica entre seções

Criação de galeria interativa com lightbox

Tecnologias Utilizadas

HTML5
Responsável pela estrutura da página e organização do conteúdo.

CSS3
Utilizado para estilização, layout, animações e efeitos visuais.

JavaScript
Responsável pela interatividade da página, incluindo troca de seções e funcionamento da galeria.

Google Fonts
Fontes utilizadas no projeto:

Cinzel

EB Garamond

Estrutura do Projeto

O projeto é dividido em três arquivos principais:

index.html
Contém toda a estrutura da página, incluindo seções, textos, imagens e navegação.

style.css
Arquivo responsável por toda a estilização visual do site, incluindo cores, layout, animações e responsividade.

script.js
Responsável pelas funcionalidades interativas do site.

Estrutura da Página

A página possui as seguintes seções principais:

Header

Contém o título principal "Lua Nova" e uma citação marcante do personagem Edward Cullen sobre Bella Swan.

O header também utiliza efeitos visuais com gradientes e brilho que simulam a estética de uma noite iluminada pela lua.

Menu de Navegação

O menu permite navegar entre as diferentes seções do site:

Início

Resumo

Protagonista

Personagens

Galeria

A navegação é controlada por JavaScript, que ativa ou oculta cada seção conforme o botão selecionado.

Seção Início

Apresenta uma introdução sobre a história de New Moon, explicando os acontecimentos após o aniversário de Bella e a decisão de Edward de se afastar para protegê-la.

Seção Resumo

Mostra um resumo da trama, destacando o momento em que Bella enfrenta a separação de Edward e encontra apoio na amizade com Jacob Black.

Também inclui um GIF ilustrativo relacionado à história.

Seção Protagonista

Apresenta informações sobre a personagem principal:

Bella Swan

A seção descreve suas características físicas, personalidade e os desafios emocionais que enfrenta durante a narrativa.

Seção Personagens

Apresenta alguns personagens importantes da história:

Edward Cullen
Vampiro que ama Bella e toma a difícil decisão de se afastar para protegê-la.

Jacob Black
Melhor amigo de Bella que se torna seu apoio durante um período difícil.

Alice Cullen
Vampira otimista que cria um forte laço de amizade com Bella.

Cada personagem possui:

imagem ilustrativa

nome

descrição

Galeria

A seção galeria apresenta várias imagens animadas (GIFs) relacionadas ao universo da história.

As imagens possuem efeitos de:

zoom ao passar o mouse

sombras

bordas estilizadas

Ao clicar em uma imagem, ela é ampliada através de um lightbox, permitindo visualização em tela cheia.
